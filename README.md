## About

Simple Pomodoro is a web application designed to help you manage your time effectively. Built with modern web technologies, it features a clean interface that minimizes distractions, allowing you to focus on what matters most.

## Features

- **Focus Timer** - Standard 50-minute work sessions to maximize deep work.
- **Minimalist Design** - A distraction-free interface with a soothing color palette.
- **Visual Feedback** - Clear visual cues for timer status.
- **Responsive** - Optimized for both desktop and mobile devices.

## Tech Stack

- **React** - UI Library
- **TypeScript** - Static Typing
- **Tailwind CSS** - Utility-first CSS Framework
- **shadcn/ui** - Reusable UI Components
- **Vite** - Next Generation Frontend Tooling

## Getting Started

Follow these steps to run the project locally.

### Prerequisites

Ensure you have Node.js or Bun installed on your machine.

### Installation

1. Clone the repository

   ```bash
   git clone https://github.com/andre4383/SimplePomodoro.git
   ```

2. Navigate to the project directory

   ```bash
   cd SimplePomodoro
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Start the development server
   ```bash
   npm run dev
   ```
